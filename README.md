# XHappySpeed
> Download the latest Release [@here](https://github.com/Crystopia/XHappySpeed/releases)<br />

Simple plugin for Minecraft that enhances the Happy Ghasts added in the 1.21.10. 

## Features
- Speed Managment with /happy speed
- Passenger Management with /happy passenger (owner based)
- Parking System with RIGHT CLICK.

- Config.yml for enabling and changing settings from the Features
- Every Message ist customizable with MiniMessage Support.

## Versions
**Tested**: 1.21.10<br />
**Support**: ``1.21.10`` - ``26.1``
